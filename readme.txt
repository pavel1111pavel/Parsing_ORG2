Данная работа представляет собой парсер, разработанный с использованием Selenium, инструмента для автоматизации веб-браузера. Парсер предназначен для сбора информации о ценах и наличии товаров с веб-страницы. С помощью Selenium мы можем автоматически выполнять действия на веб-странице, такие как заполнение форм, нажатие кнопок и получение данных. Результаты парсинга сохраняются в формате Excel для удобства последующей обработки.
